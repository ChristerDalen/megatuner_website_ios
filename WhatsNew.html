    Version 1.8 (July 8, 2023)

      Improvements:
    • Improved efficiency of the underlying PID Controller tuning routine.
    • Added support for importing discrete state space models, allowing users to paste and utilize models from various sources, such as 1st principles.
    • Importing (DAQ tab) txt-files are now supported.
    • Expanded language support with the addition of Arabic, Chinese, Hindi, Russian, Japanese, and Spanish translations.
    • The data entered into the DCS setup will not reset each time you dismiss the sheet.

    Version 1.7 (May 15, 2023)

      Improvements:
    • Implemented a change to the closed loop data chart functionality. This change ensures that the closed loop data effectively fills out the chart.
    • Improved the user interface by introducing conditional button availability. This modification prevents users from initiating actions without appropriate input.
